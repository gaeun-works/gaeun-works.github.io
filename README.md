# 허가은 포트폴리오 사이트

퍼널 전 단계의 메시지를 설계하고, 데이터로 직접 검증하는 마케터 허가은의 포트폴리오 정적 사이트입니다.

## 파일 구조

```
gaeun-works.github.io/
├── index.html          # 메인 HTML (포트폴리오 전체 내용)
├── style.css           # 스타일시트 (다크모드, 애니메이션)
├── script.js           # 인터랙션 및 애니메이션 스크립트
├── resume_허가은.pdf   # 이력서 원본
└── 허가은_포트폴리오_최종.pdf  # 포트폴리오 원본
```

## GitHub Pages 배포

1. GitHub에서 `gaeun-works.github.io` 저장소 생성
2. 이 폴더의 파일을 push
3. Settings → Pages → Branch: main, /(root) 설정
4. `https://gaeun-works.github.io` 로 접속 가능
